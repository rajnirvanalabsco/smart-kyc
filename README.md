# Smart-KYC
An opensource KYC web application



### Features

* Easily customizable
* Usability and scalibility.
* Powerful admin dashboard and management tools.
* Highly configurable and extendable.
* Free and open-source.

### Requirements

* Linux / Mac OSX
* PHP 5.6 +
* Git 1.7.10+
* Redis 2.0+
* MySQL
